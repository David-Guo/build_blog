# Flask web 框架学习

学习使用Flask 框架构建简单的博客

[开发日志](./buildlog.md)

## 环境

1. 系统：Ubuntu 14.04 64
2. 虚拟环境：virtualenv


## 安装

下载工程并开启虚拟环境

```shell
git clone https://github.com/David-Guo/build_blog
cd build_blog
``````

启动虚拟环境，安装工程依赖

```shell
$ virtualenv venv
$ source venv/bin/activate
(venv)$ pip install -r requirements.txt
``````

退出虚拟环境

```shell
(venv)$ deactivate
``````
